\modHeadsection{スピンドル回転数の基本事項\TBW}
\begin{enumerate}
\item \index{でんげんとうにゅうじ（きかいほんたい）@電源投入時（機械本体）}電源投入時の\index{しゅじくかいてんすう@主軸回転数}主軸回転数：0
\item 設定可能な最小単位：1回転/min
\end{enumerate}
