\tPart{幾何学・運動学的特性の解析計算}{概要}{%
ソフトウェアの観点から見た場合、高精度な加工を実現するためにはモールドや工具の幾何学的特性、さらにはマシニング内での運動学的特性を正確に把握することが不可欠であることは言うまでもない。
マシニングセンタの操作・指示はこうした特性の理解が前提である。
さらに、CAD, CAMを利用するというのであれば、これに加えてCADによる3Dモデリングの技術、およびそれに伴うCAMの操作技術も習熟しなければならない。

　ここでは、まず最初の段階として、モールドや工具等の幾何学的特性およびマシニング内での運動学的制約を考慮し、加工の際に必要となる位置・距離等の解析的な導出を試みる。
}
\input{./MACN_part_Analytical_Calculation}

\tPart{解析計算に基づく数値解析}{概要}{%
前段階では、製品や工具等の幾何学的・運動学的特性の正確な把握を試みるため、寸法等を一般化した上で解析的な計算・導出を行った。
一方、個々の製品や工具等はすべて固有の寸法等を持つ。
そのため実際の加工（プログラムの作成）の際には、それら全てに対して具体的な数値情報を指定する必要がある。
こうした数値情報は製品ごとに膨大にあり、このような手続きを手作業で行うことは非現実的である。
実際、品質はもちろんのこと、作業効率・人的資源・安全(security)・安全(both physical and mental safety)・保守など、いずれの観点からみても能率が著しく低下するのは自明である。

　したがって、こうした手続きを可能な限り自動化することが必須事項である。
それに伴い、ここでは属人性を極力排除する形で必要な数値情報が得られるシステムの提供を試みる。
}
\input{./MACN_part_Numerical_Calculation}
