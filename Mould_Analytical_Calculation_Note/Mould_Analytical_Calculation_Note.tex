\tPart{幾何学的な解析計算}{概要}{%
マシニングを用いた加工の特徴として、その精度の高さが挙げられる。
ソフトウェアの観点から見た場合、高精度な加工を実現するためにはモールドや工具の幾何学的特性、さらにはマシニング内での運動学的特性を正確に把握することが不可欠であることは言うまでもない。
しかし、現状ではこうしたソフトウェアの観点から見た特性の把握はほとんどなされていない。

　このような背景から、まず最初の段階として、モールドや工具等の幾何学的特性およびマシニング内での運動学的制約を考慮し、加工の際に必要となる位置・距離等の解析的な導出を試みる。
}
\input{./MACN_part_Analytical_Calculation}

\tPart{諸規程 および 諸標準の策定}{概要}{%
現時点(\customtoday 時点)における客観的事実として、番号付けに関する業務規程・技術標準等は存在しておらず、もとより「ソフトウェア関連について管理・業務を行う部門」は存在すらしていない状態である。
また、同じく客観的な事実として、このような状態は旧態依然として永らく続いており、今後もこれが変化する可能性は極めて低いと言わざるを得ない状況にある
%% footnote %%%%%%%%%%%%%%%%%%%%%
\footnote{言わずもがなではあるが、これは第二製造室に限った話ではなく、社内全体の状態である。}。

　こうした事情から、\DMname（特にソフトウェアの観点）について、独自に規程および標準の策定・制定を行う。
なお、ここに記載しているものは正式なルールではなく、だいたいの目安・方針である。
}
